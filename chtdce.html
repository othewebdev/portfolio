Description:


Chatidice is a chat application built using NodeJS, Express, and Socket.io. You can choose between 3 rooms, and add an unlimited amount of users!


Bugs I Ran Into:

  • USERNAME did not display correctly on message enter
  • Client crashed when 3 or more users joined a room
  • USERS did not display in a list correctly in the sidebar panel

How I Fixed Them:

  • Exported USERNAME function as module and imported to entry point (server.js)
  • Re-structured ADD USERS function
  • Re-structured ADD USERS function, added new DOM element to main.js
