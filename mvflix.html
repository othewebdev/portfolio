Description:


Movieflix is a Netflix clone site built to demonstrate a full stack ReactJS involved web application. This website allows the user to create an account, sign in, and browse selections. Web application deployed through Firebase. Backend data containing list of films and series is also stored in Firebase. Web components constructed with both styled and compound components. Application designed to redirect user through pages via React Router. Auth verifies if user is in database before redirecting. Fuse.JS used for live search capability on the browse page.


Bugs I Ran Into:

  • Trouble installing Firebase dependency through NPM
  • Google Plus+ accounts may get stuck loading after sign in.

How I Fixed Them:

  • Added Firebase directly to package.json then ran "npm install"
  • No fix- this is an issue with both Firebase and Google Plus.
