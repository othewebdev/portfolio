Description:


The Netvink website is a SPA ReactJS web app that uses styled components. Pages are loaded dynamically in a separate module file. These pages are stored as objects. React Router was used to create the Sign-In page, aswell as React Scroll for the homepage elements.


Bugs I Ran Into:

  • React Scripts would not initially allow server to start without significant errors.
  • Page did not display after being deployed to hosting client

How I Fixed Them:

  • React Scripts rolled back to React Scripts 2 until OCT 04, 2020 due to an error with React Scripts 3.
  • React Router needed to be configured with basename in order for site to load on custom domain.
