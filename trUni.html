Description:


Truley Unique is a site created for an Etsy shop business. The client wanted a simple, clean, and efficient way to navigate current product selections. Products needed their own respective pages, the product page itself needed a search function to filter selections in respective categories. TailwindCSS and jQuery were used in majority to implement stick nav, search filter, and mobile responsiveness. Additional requirements included ability to host in an S3 bucket, and Paypal legacy button integration.


This is an on going project that is always being maintained. Bugs are continous and never-ending!
